# セキュリティ要点

このアプリは少人数チーム向けの出欠管理用です。個人情報を最小限にし、以下の前提で使ってください。

- 推奨用途: 練習日 / 練習時間 / 出席・欠席のみ
- 推奨しない用途: 住所、電話番号、病歴、決済情報などの保存
- 通信は公開URL側で HTTPS を有効にする
- `SECURE_COOKIES=1` を設定する
- PIN は定期的に変更する
- 管理者PINはメンバーPINと別管理にする
- 不要な共有URLの拡散を避ける

実装済み対策
- PIN のサーバー側ハッシュ化保存
- サーバー側で権限制御
- セッションCookie (HttpOnly, SameSite=Lax)
- ログイン失敗回数の簡易制限

残る注意点
- 企業向けSSOや多要素認証ほど強固ではありません
- PINを知っている人はそのアカウントとして入れます
- 公開先サービス自体の設定ミスは別途注意が必要です

## v4 の補足
- 登録者追加は DB に保存されます。
- 練習日程の編集はサーバー側で更新されます。
- 既存の初期PINを変更したい場合は、generate_config.py で config.json を更新してください。
